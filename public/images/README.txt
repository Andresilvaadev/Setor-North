COLOQUE SUAS FOTOS AQUI
========================

Estrutura esperada:

  products/   -> fotos dos produtos do catálogo
                 (use o mesmo nome do campo "image" em src/data/siteContent.js)
                 ex.: sn-cam-001.jpg, sn-mol-001.jpg ...

  lookbook/   -> fotos de looks (look-01.jpg, look-02.jpg, look-03.jpg)

  hero/       -> imagem de fundo opcional da capa

  dtf/        -> imagens da seção de personalização (opcional)

Dica: enquanto não tiver foto, o site mostra um placeholder com o ID do produto.
Formato recomendado: JPG, proporção 4:5 para produtos (ex.: 1000x1250px).
